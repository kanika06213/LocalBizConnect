# LocalBizConnect

LocalBizConnect is a marketplace platform that connects local businesses with nearby customers.

Users can browse categories, explore local shops, view products/services, and place orders.

## Features

### Customer
- Browse service categories
- View shops in each category
- Explore shop products
- Add products to cart

### Vendor
- Vendor dashboard
- Add and manage products
- View customer orders

### Admin
- Admin dashboard
- Manage shops
- Manage users
- View all orders

## Tech Stack

- React.js
- React Router
- CSS
- Node.js (Backend integration coming)

## Future Improvements

- Authentication system
